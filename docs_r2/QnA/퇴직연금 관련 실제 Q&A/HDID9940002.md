# 전산입력방법

> 수신 > 등록·해제 > 일반등록·해제 [20]

계좌번호[연금신탁 ]등해코드검색 [21102-신탁계좌 적립기간 변경등록]
변경후 만기일[ ]
> ※2013년 소득세법령 개정으로 구 연금신탁의최소 납입기간이 '10년 이상'에서→'5년 이상'으로단축되었음
> ※기존연금신탁 계약기간 단축 가능여부
 만 55세가 되고 연금저축기간이 5년이 경과한 계좌라면월단위로계약기간 축소 가능
> (만 55세가 되고 5년을 불입했다면 연금 수령개시 가능)
