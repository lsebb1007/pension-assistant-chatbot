# Q&A
## 1 고객이 퇴직금을 비대면으로 상환하면 영업점에서 지급정지 해제 처리 가능한가요?
네. 가능합니다.
퇴직금 입금 시 자동등록 되는 지급정지 건은 영업점에서 기대출 상환사실 확인 후 해제 처리 가능합니다.
