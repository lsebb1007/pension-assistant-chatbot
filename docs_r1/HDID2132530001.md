## 2 A회사의 퇴직연금 가입자가 계열사인 B회사로 전직 시 계약이전 가능여부
1)B회사에서A회사에서의근속년수 불인정시
→퇴직 및 의무이전[개인형IRP]
처리
2)B회사에서A회사에서의근속년수 인정시
→계약이전처리
예) DB(A회사) -> DB(B회사) : 가능/ DC(A회사) -> DB(B회사) :불가
DB(A회사) -> DC(B회사) : 가능/ DC(A회사) -> DC(B회사) : 가능
