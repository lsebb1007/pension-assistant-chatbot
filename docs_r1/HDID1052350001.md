# 참고사항
1.연금신탁을 인터넷에서 신규한 경우
신규 당시의 유동성출금계좌가연결계좌로등록되어,인터넷에서직접 계좌해지시 연결계좌로 해지대금이 입금되도록 설정됨
2.위와 같이 연결계좌로 등록된 유동성계좌는 연결계좌 해제 처리전에 계좌해지가 불가함
3.연금신탁은 적립만기일 이후 연금으로 지급받는 상품이므로, 적립만기일에 자동해지 처리는 되지 않음
