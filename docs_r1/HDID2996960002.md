# 가입자부담금 입금분에 대한 퇴직연금납입확인서 출력 전산입력방법
## 전산업무 참고정보
> 1.가입자부담금 대해 세액공제 받기 위해 발급하는 확인 서류임☞발급 대상 :가입자 부담금이 있는 DC가입자/기업형IRP가입자/개인형IRP 가입자만 발급 가능
> 2. 발급 채널 : 영업점, 개인인터넷뱅킹,국세청 홈택스
> ☞ 해지 계좌는 비대면 발급 불가(영업점 방문 발급)
> ☞ 신한 쏠(SOL) 발급 불가
> 3. 영업점 발급시 NET점거래 가능
