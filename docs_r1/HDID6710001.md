# 상품별 유의사항
## 1 정기성
1. 정기성 상품의이자지급 및 원가지급은모두거래구분[12-이자지급]으로거래 가능
2.정기성 연금지급식상품의연금지급은거래구분[13-연금지급]으로거래 가능
3. 구조흥의 미지급이자 정산도 거래구분[12-이자지급]으로 거래 가능
4. 정산되지 않은 미지급이자와 지급할 이자가 있는 경우 이자지급거래로 미지급정산 및 이자지급이동시에처리됨
