# 연금저축펀드「납입금 전환 특례 제도」
1. 2014년 소득세법 시행령 개정에 따른 적립금 전환 특례 제도 시행
☞연금계좌
소득·세액공제
한도 초과납입금 또는 소득·세액공제 미신청 납입금을당해 연도의 납입금으로전환할경우 세액공제가 가능함☞ 당행 사원복지연금계좌 전환거래 불가
