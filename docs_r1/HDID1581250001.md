# 연금(저축)신탁 일부해지(인출) 전산처리방법
1.가입기간년도 별공제한도 초과납입금 또는 소득·세액공제 미 신청 납입금에 대해일부해지(인출) 가능함(연금저축신탁, 연금신탁 두가지 상품 모두 적용)
2. 타기관 사원복지연금 계좌일부해지 불가함
3. 당행 임직원 사원복지연금일부인출 가능함(21.1.4일 변경)
4.ISA 만기자금 연금 전환 입금계좌 일부해지 가능 함
